# 🥛 Reddy Premium Dairy

A full-featured dairy e-commerce platform built with Next.js 16, featuring OTP authentication, admin dashboard, order management, and more.

## 🌐 Live Website

**➡️ https://reddy-primum-dairy.vercel.app**

## ✨ Features

- 🛒 Product catalogue with cart & wishlist
- 🔐 OTP-based registration (Email / SMS)
- 📦 Order tracking with real-time status
- 🧑‍💼 Admin dashboard (analytics, products, orders, payments, delivery)
- 💳 Multiple payment methods (COD, UPI, Card)
- 🌙 Dark mode support
- 📱 Mobile responsive

## 🚀 Getting Started (Local Development)

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it locally.

## 🛠️ Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Database:** MongoDB Atlas + Local JSON fallback
- **Auth:** OTP via Email (Nodemailer) / SMS (Twilio)
- **Styling:** Tailwind CSS v4
- **Deployment:** Vercel
